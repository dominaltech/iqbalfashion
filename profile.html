Merchant/Developer: Setting up Google Auth in Supabase
  1. Go to the Google Cloud Console, create a project, and navigate to APIs & Services > Credentials.
  2. Create an OAuth 2.0 Client ID for a web application.
  3. Copy your Supabase project's Redirect URI (from Supabase Dashboard > Auth > Providers > Google). Paste this into Google Cloud's Authorized Redirect URIs.
  4. Copy the generated Client ID and Client Secret from Google.
  5. Go to your Supabase Auth Panel > Google provider, enable it, paste the Client ID & Secret, and save.
